(defun jh-editing/init-affe ()
  (use-package affe
    :after consult orderless
    :demand t
    :config
    ;; Use orderless to compile regexps
    (defun +affe-orderless-regexp-compiler (input _type _ignorecase)
      (setq input (orderless-pattern-compiler input))
      (cons input (lambda (str) (orderless--highlight input str))))

    (setq affe-regexp-compiler #'+affe-orderless-regexp-compiler)

    (global-set-key (kbd "M-s a") 'affe-grep)
    (global-set-key (kbd "M-s A") 'affe-find)
    (spacemacs/set-leader-keys "sa" 'affe-grep)
    (spacemacs/set-leader-keys "sA" 'affe-find)

    ;; Manual preview keys
    (consult-customize affe-grep affe-find
                       :preview-key '("M-n" "M-p"))
    ))

    Puni Configurations #

    There’s a lot of really good editing tools. Puni is a more simple and modern approach to structured editing, so let’s use that. I used to use smartparens, but it had a lot of issues where expressions becoming unbalanced in strict mode, which is annoying, because then you have to get out of strict-mode to fix it.
